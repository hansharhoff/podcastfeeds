How to use

  • Listen: subscribe to a feed URL above in Pocket Casts (Discover → paste URL in search). Subscribe to either "All feeds" or individual feeds — both gives duplicates.
  • Share an article from the phone: paste its URL in the form above, or install the HTTP Shortcuts app and create a shortcut with method POST, URL {{ base }}/{{ token }}/api/submit, request body type form parameters, one parameter url = {{share_text}} (a "share text" variable), then enable "Allow sharing to this shortcut". Articles land in the Inbox feed a few minutes later.
  • Add/remove sources: edit config/sources.yaml in the repo and docker compose restart. Types: rss (each post = episode), digest (many items → one rephrased episode on a cron schedule).
  • Voices: leave voice empty for an automatic, permanent per-source/blogger voice; quotes inside articles are read by a second voice. Set voice to fix one (news = en-GB-RyanNeural, Home Assistant = en-GB-ThomasNeural).
  • Images: narration announces figures; the images appear in the episode show notes and as chapter art (tap the artwork / chapter list while playing).
  • Errors: failed episodes show a retry button in the list below. Logs: docker compose logs -f.
